# COMP 3020 Project

## Requirements

- **Node.js**: v20.15.0
- **npm**: v10.8.1

### Installing Node.js and npm

If you donâ€™t have Node.js and npm installed, download them from the Node.js website. This will install both Node.js and npm.

Verify Installation:

```bash
node -v   # Should output v20.15.0
npm -v    # Should output v10.8.1
```

## Getting Started

1. **Clone the Repository**
    ```bash
    git clone https://github.com/ron12341/comp3020-project.git
    cd winnipeg-clothes
    ```
   
2. **Install dependencies:** Use the following command to install all required dependencies:

    ```bash
    npm install
    ```

## Running the project

To start the project in development mode:
```bash
npm run dev
```

Vite will launch the website at http://localhost:5173/
